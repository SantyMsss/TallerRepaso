# TallerRepaso — Capítulo 1 y Capítulo 2 (Git)

Breve documentación para el proyecto de repaso sobre Git y GitHub.

## Descripción
Proyecto con una guía básica en HTML sobre conceptos fundamentales de Git (Capítulo 1) y una página adicional muy básica para Capítulo 2. Incluye un tema en modo oscuro y navegación entre las páginas.

## Estructura de archivos
- `index.html` — Capítulo 1 (modo oscuro) con botón que redirige a `capitulo2.html`.
- `capitulo2.html` — Página básica del Capítulo 2.
- `README.md` — Documentación (este archivo).

## Cómo probar localmente
Opción 1: Abrir `index.html` directamente en el navegador.

  ```
  Luego abrir: `http://localhost:8000/index.html`

## Cambios principales incluidos
- Tema en modo oscuro aplicado a `index.html` y `capitulo2.html`.
- Botón en `index.html` que enlaza a `capitulo2.html`.
- Página nueva `capitulo2.html` con contenido básico.

```
